Source: Precios de combustibles - MinEnergía.

Se realiza un análisis del What?, el Why? y el How? propuestos en el framework de Tamara para diseñar una visualización.

Ésta viz permite analizar cómo varían el precio promedio de combustibles por departamento en Colombia.

Tareas en el dominio del contexto:

  • T1(Principal): Presentar cómo varía mensualmente el precio promedio de combustibles en todos los departamentos de Colombia
  • T2(Secundaria): Comparar entre departamentos la variación mensual del precio promedio de combustibles
  • T3(Secundaria): Identificar patrones asociados a la variación mensual del precio de combustibles.

What?

Se tiene un dataset de tipo temporal, donde cada ítem corresponde a información de precio de combustibles, asociada a un departamento en un mes específico

  • Categóricos: Departamento, Municipio, Nombre Comercial, Bandera, Producto
  • Ordinales:Fecha Registro, Periodo. Adicionalmente, a partir del campo Periodo se calcula el campo Mes extrayendo el año y el mes.
  • Cuantitativos secuenciales: Precio. Adicionalmente, a patir del campo Precio se calcula el campo Precio Promedio (agregando por Departamento).
  • Cuantitativos divergentes: A partir de Precio Promedio se calcula el campo Variación (variación en porcentaje respecto al mes anterior )
  • Para la visualización se usaron las variables Departamento, Mes, Precio Promedio y Variación.

Why?

  • T1(Action:Present, Target:Trend) - Para una variable categórica (Departamento), presentar como varía en el tiempo una variable cuantitativa divergente(Variación).
  • T2(Action:Compare, Target:Trend) - Para una variable categórica (Departamento), comparar como varía en el tiempo una variable cuantitativa divergente (Variación).
  • T3(Action:Identify, Target:Features) - Para una variable categórica (Departamento), identificar patrones en la variación de una varibale cuantitativa divergente(Variación)

How?

Encode

Map
  • Mark: Área
  • Channel: Posición en x para Mes
  • Channel: Spatial region en y para Departamento
  • Channe: Length y para valor de Variación
  • Channel: Color hue (orange para aumento y green para disminución)
  • Channel: Color saturation (a mayor valor de Variación mayor saturación)
Idiom

Horizon chart separate, order and align

Comentarios

Para una persona del común, puede ser interesante descubrir que el precio de los combustibles puede aumentar en algunos departamentos mientras en otros disminuye.

Sin embargo, se puede ver que en algunos meses como mayo de 2016, octubre de 2017, septiembre y octubre 2018, el precio promedio de los combustibles aumentó en todos los departamentos, en mayor proporción en algunos casos.

Adicionalmente, se observa un aumento grande en el precio promedio de combustibles entre junio y julio de 2019

La viz podría mejorarse incluyendo otras tareas de análisis, desagregando por tipo de combustible (producto), empresa(bandera) o municipio.
