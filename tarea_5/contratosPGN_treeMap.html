Source: Listado de los 100 contratos más grandes con recursos del PGN - 2019

Se realiza un análisis del What?, el Why? y el How? propuestos en el framework de Tamara para diseñar una visualización.

Ésta viz permite analizar (por valor) cuales sectores y beneficiaros tienen mayor participación, dentro de los 100 contratos más grandes, realizados con el Presupuesto General de la Nación (PGN) en 2019.

Tareas en el dominio del contexto:

  • T1(Principal): Presentar la distribución por sectores y beneficiarios de los 100 contratos más grandes realizados con el PGN 2019
  • T2(Secundaria): Indetificar el sector con mayor participación
  • T3(Secundaria): Indetificar el beneficiario con mayor participación dentro de su sector
  • T4(Secundaria): Indetificar el beneficiario con mayor participación dentro dentro del total de contratos

What?

Se tiene un dataset de tipo Tree representado como table(no se usarán los campos temporales), donde cada ítem corresponde a información de un contrado realizado

  • Categóricos: Nombre Sector, Sub Unidad, Entidad, Beneficiario,
  • Ordinales:Fecha Registro
  • Cuantitativos secuenciales: Valor Contrato, Valor Pagado y %Pagado. Además, a partir de Valor Contrato se derivan los campos Valor Sector, % dentro del Sector, Valor Total y % dentro del Total
  • Para la visualización se usaron las variables Nombre Sector, Beneficiario, Valor Contrato, Valor Sector, % dentro del Sector, Valor Total y % dentro del Total

Why?

  • T1(Action:Present, Target:Distribution) - Para dos variables categóricas (Sector, Beneficiario), presentar como se distribuyen a partir de una variable cuantitativa secuencial (Valor Contrato).
  • T2(Action:Indentify, Target:Extremes) - Para una variable categórica (Sector), identificar cual categoría concentra el mayor valor asociado de una variable cuantitativa (Valor Contrato).
  • T3(Action:Identify, Target:Extremes) - Para una variable categórica (Benficiario), identificar cual categoría concentra el mayor valor asociado de una variable cuantitativa (Valor Contrato) dentro un sector.
  • T3(Action:Identify, Target:Extremes) - Para una variable categórica (Benficiario), identificar cual categoría concentra el mayor valor asociado de una variable cuantitativa de (Valor Contrato) dentro del total de contratos .

How?

Encode

Map
  • Mark: Área
  • Channel: Color saturation para Valor Contrato
  • Channel: Spatial region para Sector y Beneficiario
  • Channel: Area(2D size) para Valor Contrato
Idiom

Treemap

Comentarios

Para una persona del común, puede ser interante conocer cómo se ejecutan los recursos del PGN una vez se reparte el dinero en en los diferentes sectores

En el sector transporte se observa una mayor concentración de contratos, razón por la cuál es el sector con mayor valor de contratos realizados. Sin embargo, el beneficiario más grande solamente alcanza el 2,5% del valor total de los 100 contratos

El sector educación es el segundo más grande y además contiene al beneficiario más grande (Fiduprevisora S.A.) con el 14,5% del valor total de contratos. También contiene al tercer beneficiaro más grande (ICETEX) con el 6,2%.
