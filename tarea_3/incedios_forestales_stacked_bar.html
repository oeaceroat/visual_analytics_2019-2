By Oscar Eduardo Acero Acero.
Source: Datos de los incendios forestales ocurridos en el departamento de Cundinamarca.

Se realiza un análisis del What?, el Why? y el How? propuestos en el framework de Tamara para diseñar una visualización.

La anterior viz permite hacer una revisón de los tipos de terreno más afectados por incendios forestales en Cundinamarca, además de identificar cuáles cuáles son los municipios con mayor cantidad de hectáreas comprometidas.

Tareas en el dominio del contexto:

  • T1(Principal): Descubrir el tipo de terreno más afectado por incendios forestales en Cundinamarca
  • T2(Secundaria): Identificar los municipios con mayor afectada por incendios forestales
  • T3(Secundaria): Buscar por municipio la cantidad de hectáreas afectadas por tipo de terreno.

What?

Se tiene un dataset de tipo table (no se usan los campos fecha), donde cada ítem corresponde a un evento de incendio forestal reportado:

  • Categóricos: Municipio, Corregimiento, Superficial, Subterráneo
  • Ordinales:Fecha de reporte, Fecha de terminación, Hora de terminación
  • Cuantitativos secuenciales: Bosque nativo denso, Bosque intervenido, Bosque plantado, Bosque seco, Cultivos, Páramos, Sabanas y pastizales, Pastos mejorados, Rastrojo, Área total. Adicionalmente, se calculó la variable derivada "Otros" donde se agrupan los tipo de terreno con menos del 5% de observaciones
  • Para la visualización se usaron las variables Municipio, Área total, Bosque nativo denso, Sabanas y pastizales, Pastos mejorados, Rastrojo y Otros.

Why?

  • T1(Action:Discover, Target:Extreme) - Para un conjunto de variables cuantitativas (tipos de terreno), descubrir cual tiene el valor(hectáreas) global más alto.
  • T2(Action:Identify, Target:Extremes) - Para una variable categórica (municipio), identificar los n ítems con mayor valor asociado a una variable cuantitativa secuencial (área total).
  • T3(Action:Browse, Target:Value) - Para una variable categórica (municipio), buscar los valores asociados a múltiples variables cuantitativas secuenciales (tipos de terreno)

How?

Encode

Map
  • Mark: Line
  • Channel: Spatial region en y para municipio
  • Channel: Length en x para cantidad de hectáreas
  • Channel: Color hue para tipo de terreno
Idiom

Stacked bar chart separate, order and align

Comentarios

Analizando más allá del insight descrito en el encabezado,se encuentra que a pesar de que a nivel global hay una mayor afectación en terrenos de sabanas, pastizales y bosque nativo, a nivel de cada municipio no siempre se mantiene ese comportamiento.

La visualización podría mejorarse, agregando elementos que permitan descubrir alguna relación o patrón entre los municipios afectados; como ubicación geográfica, clima, actividad económica, entre otros. También podría indicarse la cantidad de incendios reportados para cada tipo de terreno, así como un resumen de las cifras globales.
